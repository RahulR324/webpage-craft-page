## HEAVEN OF CRAFT

A landing page for a handcrafted goods brand built with HTML, CSS, and vanilla JavaScript.


## Project Structure

```
webpage-craft-page/
├── index.html          # Main HTML file
├── ecommerce.css       # Styling
├── ecommerce.js        # JavaScript functionality
├── README.md           # Project documentation
└── assets/             # Images and other assets
```

## Features

- Responsive e-commerce interface
- Interactive JavaScript functionality
- Clean and organized CSS styling

## Getting Started

1. Open `index.html` in your web browser
2. Customize styles in `ecommerce.css`
3. Add functionality in `ecommerce.js`

## Files

- **index.html** - Main page structure
- **ecommerce.css** - Page styling and layout
- **ecommerce.js** - Interactive features and logic
- **assets/** - Project resources

## Usage

Simply open the HTML file in a browser to view the webpage.
